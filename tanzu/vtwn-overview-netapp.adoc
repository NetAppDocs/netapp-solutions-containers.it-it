[role="lead"]
NetApp dispone di diverse piattaforme di storage qualificate con Trident e Trident Protect per fornire, proteggere e gestire i dati per le applicazioni containerizzate, contribuendo così a definire e massimizzare la produttività DevOps.

[role="normal"]
NetApp dispone di diverse piattaforme di storage qualificate con Trident per fornire, proteggere e gestire i dati per le applicazioni containerizzate.

image:redhat-openshift-043.png["Figura che mostra il dialogo di input/output o che rappresenta il contenuto scritto"]

* I sistemi AFF e FAS eseguono NetApp ONTAP e forniscono storage sia per casi d'uso basati su file (NFS) sia per casi d'uso basati su blocchi (iSCSI).
* Cloud Volumes ONTAP e ONTAP Select offrono gli stessi vantaggi rispettivamente nel cloud e nello spazio virtuale.
* Google Cloud NetApp Volumes (AWS/GCP) e Azure NetApp Files forniscono storage basato su file nel cloud.



NOTE: Ogni sistema di storage del portfolio NetApp può semplificare sia la gestione dei dati che il loro spostamento tra siti on-premise e il cloud, in modo che i dati siano sempre disponibili dove si trovano le applicazioni.
